Student Tech Portfolio — Project Overview

Prepared by Luis Betancourt, UNC Charlotte — ITIS 3135

1. Project Summary

This project is a personal portfolio website for UNC Charlotte student Luis Betancourt. The site presents academic background, technical skills, project work, online resources, and a way for visitors to contact the student. Its purpose is to support internship and job applications while demonstrating the planning, design, and development of a modern, accessible, multi-page web application.

Intended Users

  • Recruiters and hiring managers
  • Classmates and instructors
  • Potential collaborators
  • Anyone viewing the student’s technical work

Content Overview

  • Home page with intro and featured project cards
  • Projects page with search/filter functionality
  • About page using jQuery UI tabs to organize skills
  • Resources page populated dynamically from JSON
  • Contact form with client-side validation

5. Page-by-Page Design

Page (File) Purpose Content Interactions Notes
Home
project/index.html		 Introduces the student and highlights main work. Intro text, project cards, navigation. Clickable cards to project links. Uses consistent header/footer layout.
Projects
project/projects.html		 Showcase of programming and design work. Cards with description, tech used, links. Search bar + filter dropdown (JavaScript). Supports recruiter browsing.
About
project/about.html		 Student background + skills. Bio, resume link, skills list. jQuery UI tabs used for structured skill display. Matches portfolio style.
Resources
project/resources.html		 Useful development and learning links. MDN, W3Schools, jQuery UI, FCC, etc. Populated dynamically from JSON file using fetch(). No database storage.
Contact
project/contact.html		 Allows visitors to send a message. Form with name/email/message. Front-end validation (JavaScript). No backend functionality.

6. Planned Dynamic Features

  • Client-side search + filtering on the Projects page.
