Student Tech Portfolio — Project Overview

Client: Luis Betancourt · UNC Charlotte — CCI · Prepared: Nov 2, 2025

Project Overview

A clean, modern portfolio site for a UNCC student that showcases skills, projects, resume, and contact info. The goal is to support internship and job recruitment and demonstrate planning, design, and development of a small web application. :contentReference[oaicite:1]{index=1}

Purpose

  • Create a modern portfolio for a UNCC student
  • Showcase skills, projects, and resume
  • Provide contact path for recruiters/collaborators
  • Demonstrate web app planning/design/dev

What it will do

  • Display project cards and skill sections
  • Include an About page + downloadable resume
  • Provide a contact form (front-end only)
  • Load resources dynamically via JSON (AJAX)
  • Use one jQuery UI widget (tabs)

Primary users: recruiters/hiring managers, classmates & instructors, potential collaborators. :contentReference[oaicite:2]{index=2}

Dynamic Functionality

  • jQuery UI Tabs on About for organizing skill sets (accessible).
  • AJAX (fetch JSON) on Resources to render link cards (read-only, no DB).
  • Client-side filtering on Projects (search + tech dropdown).
  • Client-side validation on Contact (no submission/storage).

All features are read-only per course constraints and align with the proposal. :contentReference[oaicite:5]{index=5}
