Website Evaluations

Using the CRAP design principles (Contrast, Repetition, Alignment, Proximity) to evaluate two sites.

GOV.UK Design System

This government design system site is an example of clarity and consistency.

  • Contrast: Clear type hierarchy, bold headings, minimal purposeful colors.
  • Repetition: Layout patterns, navigation, and buttons stay consistent.
  • Alignment: Strong grid and left-aligned content aid scanning.
  • Proximity: Related content is grouped, sections separated with whitespace.

Ling’s Cars

This car-leasing site is infamous for chaotic visuals and poor hierarchy.

  • Contrast: Too many clashing colors and animations competing for attention.
  • Repetition: Inconsistent patterns and abrupt style changes.
  • Alignment: Weak structure; floating elements break scanning flow.
  • Proximity: Items are crowded together, grouping is unclear.
